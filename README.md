# CourseAdvisor

The purpose of this project is to gather together reviews and ratings for non-mandatory classes in computer science.

So that students can look for input from other students when they have to select theyr courses.

## Tech

It is built with Firebase and React.

This project is based on https://codelabs.developers.google.com/codelabs/firestore-web/index.html#0

Over the time i would like to use this project to get better at CSS and experimenting with Greensock. Here I will post updates

First version of the webiste is up and running (09.july)
That is a super simple version, very basic website that uses material-ui.


## Usage

There is no need to register or sign in.
I wanted it to be as simple as it can bee.
